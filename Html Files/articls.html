The Two Sum problem asks us to determine whether there are two distinct numbers in an array that add up to a given target sum.

Problem Statement:

You are given:

  • An array of integers.
  • A target sum.

You need to check if there are two distinct numbers in the array that, when added together. equal the target sum.

Example:

  • An array of integers.
  • A target sum.

In this case, the numbers 2 and 7 add upto 9, so that the function sould return true..

  • Brute Force Approach

    Brute Force Approach (Step-By-Step):

    The brute force approach is the simplest way to solve this problem. It involves checking all pair of numbers in the array and seeing if their sum equals the target.

    Visualization with Example:

    Let's take the array: [2,7,11,15] and the target 9.

    We will try every possible combination of two numbers and check if they sum up to 9.

    Step 1: Start with the first number:

    • Pair 2 with 7: Sum = 2 = 7 = 9. This is the target! We found the pair.
    • Since we already found the pair, We can stop here and return true.

    If we'll Continue.

    Step 2: Pair 2 with 11: Sum = 2 + 11 = 13. No match

    Step 3: Pair 2 with 15: Sum = 2 + 15 = 17. No match

    Now move to the second number:

    Step 4: Pair 7 with 11: Sum = 7 + 11 = 18. No match.

    Step 5: Pair 7 with 11: Sum = 7 + 11 = 18. No match.

    Step 6: Pair 11 with 15: Sum = 11 + 15 = 26. No match.

    Since we already found a pair (2 + 7), we can return true.

    1. Nested Loop:

      • The outer loop iterates over the array using the variable i.
      • The inner loop checks all elements after i (i.e., j = i + 1) to avoid duplicate pairs.
